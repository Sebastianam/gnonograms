02. Design mode

When Gnonograms starts it is in "design" mode and displays the "design grid". This grid contains the solution to the puzzle.

In this mode you can:

  • Change the dimensions of the grid

  • Design your own puzzle by drawing on the design grid

  • Save a puzzle you designed

  • Load an existing puzzle

  • Load a partially finished puzzle saved earlier

  • Start the puzzle editor to enter clues directly or to enter the descriptive information about the puzzle that is displayed at the bottom of the window.

  • Import a suitable image file, converting it into a nonogram puzzle.

  • Let the computer generate a solvable puzzle (random pattern)

  • Switch to solving mode
