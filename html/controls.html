5. Gnonogram Mouse/Keyboard Controls

Manipulation of the grid is achieved either by using the mouse (or trackpad) or by using the keyboard.

  • Move grid cursor

    Use the mouse (trackpad) or the keyboard cursor keys (Arrow keys).

    The selected cell is marked by a rectangular coloured line and the corresponding row and column clues are highlighted.

  • Mark a cell filled

    Left click or F

  • Mark a cell empty

    Right click or E

  • Mark a cell unknown

    Left double click or X

  • Mark a block filled/empty/unknown

    1. Position cursor at one end of block

    2. Hold down the corresponding button/key

    3. Move cursor to end of block

    4. Release button/key.

  • Increase/decrease font size (and thereby the size of the whole window)

    Position the cursor within the grid. Press + to increase font size or - to decrease it.

  • To determine the block extent and freedom of a row or column

    Hover the cursor over the clue. A tooltip appears giving the required information.
