9. Word Meanings

Block

A group of contiguous (filled) cells.

Block extent

The minimum number of cells required to accomodate the blocks (with one empty space between each).

Cell

Smallest square in the grid. Can either be coloured as filled, empty or unknown

Clue

The set of numbers that indicate the pattern of filled cells in a row/column (see 1. How to Play Gnonograms)

Freedom

How much room for manoeuvre there is in a row/column. The difference between the total number of cells in the row/column and the block extent.

Region

A row or column or a length of contiguous cells within a row or column

Range (of a block)

The region within which the block must be placed

Design Grid

The rectangular array of cells that holds the solution to the puzzle or on which you can draw your own puzzle

Working Grid

The rectangular array of cells that holds your working when trying to solve a puzzle

Guide Grid

A grid of lines outlining the borders of the cells
