File Menu and Toolbar icons

Summary

The file menu provides the following functions
Create a new puzzle design*
Load puzzle designs from storage media*
Load last (unfinished) puzzle and its working grid from storage media
Save puzzle designs to storage media*
Save current (unfinished) puzzle and the working grid to storage media
Quit the program.

*These functions are also provided by the tool bar

  • New

    Empty the current grid and enter "design" mode

  • Load ...

    • Shows a file selection dialogue with the puzzles in the current game directory. By default gnonogram game files are shown but the filter can be changed to Picto games.

      If the directory is changed then that becomes the default game directory.

    • If the program was closed before finishing a puzzle (or the unfinished puzzle was manually saved) then the partially completed puzzle is reloaded and solving mode entered

    • Shows a file Save As dialogue in the current game directory. The puzzle is saved in Gnonogram format. Use the separate menu option to save in Picto format. If the directory is changed then that becomes the default game directory.

    • Store the current puzzle and grid position in a temporary file so that it can be finished later. Only one unfinished puzzle can be stored at a time.

      If a second puzzle is stored it will over-write the previous one.

  • Shows a file Save As dialogue in the current game directory. The puzzle is saved in Gnonogram format. Use the separate menu option to save in Picto format. If the directory is changed then that becomes the default game directory.

  • Saves the current configuration settings (difficulty, dimensions, colours) and closes the program
