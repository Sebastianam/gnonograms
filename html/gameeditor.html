09. Game editor

The game editor may be used to enter descriptive information concerning a game you have created yourself. Alternatively, you can enter clues from a puzzle you have found elsewhere.

The game editor is started from the "Edit puzzle" option on the "Puzzle" menu.

On selecting this menu option, the window shown below is opened:

There are three tabs:

  • Description

    >Game editor window - description tab

    The desired name of the puzzle, the name of the designer, the date created and the type of license can be entered and edited in the labelled fields.

  • Row clues

    Game editor window - row clues tab

    The clue for each row can be entered and edited in the boxes provided. The clues must consist of numbers separated by commas.

    On completion of each field the contents are checked to have the correct form and to not specify more filled cells than could fit in the row. If the clue is invalid, the background of the clue is coloured and the 'OK' button disabled. The figure below illustrates some invalid clues.

    Invalid clue formats

  • Column clues

    This tab behaves in exactly the same way as the 'Row clues' tab.

When all the information has been entered, press 'OK'. The window closes and the current puzzle is updated with the new information.

The computer then attempts to solve the puzzle to check whether the clues are consistent with each other. If not, the word "Invalid" appears against "Score:" at the bottom of the main window.
